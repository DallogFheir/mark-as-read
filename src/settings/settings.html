Mark as Read

How to use

Mark as Read is a simple extension that allows you to mark webpages as read. It will apply CSS style to all links pointing to those webpages. You can customize the CSS and the URL preprocessor below.
The extension icon in the browser toolbar tells you the state of the extension:
  • – extension is disabled
    you can enable it by clicking on it while holding the Control key
  • – the current page has been marked as read
    click on the icon to mark it as unread
    click on the icon while holding the Control key to disable the extension
  • – the current page has NOT been marked as read
    click on the icon to mark it as read
    click on the icon while holding the Control key to disable the extension
Click on the icon while holding the Shift key to open the settings page.
